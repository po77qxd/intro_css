Introduction à la Syntaxe CSS

Une règle CSS est composée de trois parties principales :

  • Sélecteur : Cible l'élément à styler.
  • Propriétés : Définissent les aspects du style à modifier.
  • Valeurs : Décrivent les détails des propriétés.
                /* Exemple de règle CSS */
                h1 { 
                color: blue;font-size: 24px;
                }
Propriétés CSS Courantes
Propriété Description Exemple
color Définit la couleur du texte. color: red;
background-color Définit la couleur de fond. background-color: yellow;
font-size Définit la taille de la police. font-size: 16px;

Réponses

Partie titre

                   color: red;
                   text-align: center;
                   text-decoration: underline;

Partie paragraphe

                  color: green;
                   font-family: arial;

Partie body

                   background-color: white;
                   color: red;

Partie footer

                    display: flex;
                   justify-content: space-between;
